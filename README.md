# FloatCursor-Beta
Extension Chrome de souris personnalisé (Version beta)

## Installation manuel
Aller sur la page :
- brave://extensions/
- chrome://extensions/
- edge://extensions/
- mxlwbrowser://extensions/
Cliquer sur le bouton "Mode développeur" et
Cliquer sur "Charger l'extension non-empaquetée"

## Version Beta
Cette version est une version de test, une prochaine arrivera bientôt dans l'organisation Maxlware Developement.
