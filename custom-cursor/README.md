# Curseurs personnalisés FloatCursor

Ce dossier contient des exemples et des modèles pour créer vos propres curseurs personnalisés pour FloatCursor.

## Structure des fichiers

- `example-cursor.css` - Exemples de CSS pour des curseurs personnalisés
- Vos propres fichiers CSS ou images peuvent être ajoutés ici

## Comment ajouter un curseur personnalisé

### Méthode 1 : CSS personnalisé

1. Créez un fichier CSS avec votre design
2. Utilisez la classe `.float-cursor` comme base
3. Ajoutez votre classe personnalisée (ex: `.float-cursor.mon-curseur`)
4. Dans l'extension, allez dans "Ajouter un curseur personnalisé"
5. Sélectionnez "CSS personnalisé"
6. Copiez-collez votre CSS et donnez un nom à votre curseur

### Méthode 2 : Images

1. Préparez votre image (PNG, SVG, JPG recommandé)
2. Taille optimale : 32x32 à 64x64 pixels
3. Utilisez un fond transparent pour les PNG
4. Dans l'extension, allez dans "Ajouter un curseur personnalisé"
5. Sélectionnez "Image"
6. Téléchargez votre image et ajustez la couleur si nécessaire

## Bonnes pratiques

### Pour les CSS :
- Gardez les animations légères
- Utilisez `currentColor` pour permettre le changement de couleur
- Évitez les propriétés CSS trop lourdes
- Testez sur différents sites web

### Pour les images :
- Compressez vos images (TinyPNG, SVGOMG)
- Utilisez le format SVG pour les designs vectoriels
- Évitez les images trop lourdes (> 50KB)
- Pensez au contraste avec les arrière-plans variés

## Partage de curseurs

Vous pouvez partager vos créations avec la communauté :
1. Exportez vos paramètres (menu principal → Exporter)
2. Partagez le fichier JSON généré
3. D'autres utilisateurs peuvent l'importer

## Dépannage

Si votre curseur ne s'affiche pas :
1. Vérifiez la console du navigateur (F12) pour les erreurs
2. Assurez-vous que votre CSS est valide
3. Vérifiez la taille des images
4. Essayez sur différents sites web
